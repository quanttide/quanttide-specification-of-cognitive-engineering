# 记忆

人类的记忆系统可以分为两大类：陈述性记忆和程序性记忆。

## 陈述性记忆

陈述性记忆（Explicit Memory）是可以用语言描述的记忆，分为三种类型：

### 事件记忆（Episodic Memory）

指个人亲身经历的、发生在特定时间和地点的具体事件。它记录的是"发生了什么"，带有情境化的细节，比如时间、地点、参与者以及当时的情感体验。  

*例子*："昨天下午在书房尝试用 opencode 开发管理后台，遇到困难，感到很挫败。"

### 语义记忆（Semantic Memory）

指客观的、去情境化的知识，包括事实、概念、规则和一般性理解。它不依赖于特定的时空背景，是关于"世界是什么"的认知。  

*例子*："opencode 适合编辑静态文本文件，但对于需要精密细节的代码编写，目前能力有限。"

### 自我记忆（Self Memory）

指与自我认知、个人身份、价值观、情感反思相关的记忆。它记录的是"我是谁"、"我怎样看待自己"以及"我的成长与变化"。  

*例子*："我一直期待自己成为顶尖程序员，但今天遇到的困难让我意识到，我的执行力与预期还有差距，这让我有些焦虑。"

## 程序性记忆

程序性记忆（Procedural Memory）是通过行为和实践获得的记忆，如技能、习惯等。
